Soil Profile¶

Configuration¶

[SOIL_PROFILE]
dzLayer = [ 0.0, 0.05, 0.15, 0.3, 0.6, 1.0, 2.0,]
dzComp = [ 0.1, 0.35, 1.0, 3.0,]

Options¶

dzLayer

Thickness of each soil layer (metre).

dzComp

Thickness of each soil compartment (metre).
