Time
====

.. currentmodule:: aquacrop

Configuration
~~~~~~~~~~~~~

.. code-block:: python

    [CLOCK]
    start_time = TODO
    end_time = TODO

Options
~~~~~~~

``start_time`` : ``str``
    The start time of the simulation, in 'YYYY-MM-DD'

``end_time`` : ``str``
    The end time of the simulation, in 'YYYY-MM-DD'
