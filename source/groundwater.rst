Groundwater
===========

.. currentmodule:: aquacrop

Configuration
~~~~~~~~~~~~~

.. code-block:: python

   [GROUNDWATER]
   water_table = false
   variable_water_table = false
   filename
   

Options
~~~~~~~

``water_table`` : ``bool``
    Whether or not the depth to groundwater should be included.

``variable_water_table`` : ``bool``
    If ``water_table = true``, this option specifies if the water table is treated as static or dynamic.
