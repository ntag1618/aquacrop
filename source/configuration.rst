Configure
=========

.. currentmodule:: aquacrop

AquaCrop-Python reads the various settings from a user-supplied TOML configuration file. The basic structure of a configuration file is as follows:

.. code-block:: python
		
    [SECTION_ONE]
    option = value

    [SECTION_TWO]
    option = value
