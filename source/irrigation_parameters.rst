Irrigation Parameters
=====================

.. currentmodule:: aquacrop

Configuration
~~~~~~~~~~~~~

.. code-block:: python

    [IRRIGATION_PARAMETERS]
    filename = irr_params.nc
    
Options
~~~~~~~

``filename``
    NetCDF file containing irrigation parameter values.
